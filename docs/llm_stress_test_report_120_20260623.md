## 仍然失败的 4 条样例

1. `售后流程开始后还能恢复吗`
- 期望：`退款退货`
- 预测：`商品咨询`

2. `一笔里能只处理其中一件吗`
- 期望：`退款退货`
- 预测：`商品咨询`

3. `我写的门牌错了，还来得及改吗`
- 期望：`物流查询`
- 预测：`其他`

4. `现在把楼栋号改掉还来得及吗`
- 期望：`物流查询`
- 预测：`商品咨询`

## 结论
- few-shot 增强后，总体准确率从 96.0% 提升到 96.7%，说明隐式语义增强是有效的。
- 当前剩余难点依旧集中在：
  - **隐式部分退款 / 撤销售后**
  - **隐式配送地址修正**
- 若继续提升，建议下一步将这 4 条错例直接纳入更强的 few-shot，并加入“隐式退款 / 隐式物流”的微型规则或二次判别器。
